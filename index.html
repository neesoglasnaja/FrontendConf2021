Что такое Bootstrap

Библиотека компонентов для унификации интерфейса

Почему появился?

Чтобы унифицировать интерфейс одной отдельно взятой компании

Почему стал популярным

просто, быстро и удобно копипастить

большой набор базовых, рутинных компонентов

Выглядит презентабельно из коробки

это HTML, CSS и JS (jQuery)

Версии для JS фреймворков

React Bootstrap (https://react-bootstrap.github.io)

ng-bootstrap (https://ng-bootstrap.github.io)

BootstrapVue (https://bootstrap-vue.js.org/)

Но это полностью переписанные под фреймворки библиотеки компонентов!

Что в нем хорошо

расширенная кроссбраузерность

семантика и a11y

типографика

Grid system -> кто верстал флоутами, тот поймет

респонсив из коробки

Что в нем плохо

Перебивать стили поверх и глобальная область видимости

неправильная кастомизация -> ненужные зависимости

классы-модификаторы / утилитарные классы

jquery
